# Health Monitoring System

## Overview
An IoT-based Health Monitoring System using ESP8266, Pulse Sensor, DS18B20 Temperature Sensor, LCD Display, Blynk, and Firebase.

## Features
- Heart Rate Monitoring
- Temperature Monitoring
- LCD Display Output
- Firebase Cloud Storage
- Blynk Monitoring
- WiFi Connectivity

## Components Used
- NodeMCU ESP8266
- Pulse Sensor
- DS18B20 Temperature Sensor
- 16x2 I2C LCD
- Blynk Platform
- Firebase

## Output
Displays temperature and BPM on LCD and uploads the data to cloud services.
